# Web Research Skill

This skill enhances your ability to perform deep web research with proper source verification and citation.

## When to Use

Use this skill when asked to:
- Research a topic comprehensively
- Find and verify information from multiple sources
- Create reports with citations
- Fact-check claims

## Research Process

1. **Initial Search**: Use `web_search` to find relevant sources
2. **Source Verification**: Visit multiple sources to verify information
3. **Cross-Reference**: Compare information across different sources
4. **Citation**: Always cite sources with URLs

## Best Practices

- Search with multiple query variations
- Prefer authoritative sources (.edu, .gov, established news)
- Note publication dates for recency
- Cross-reference facts across 2-3 sources
- Always provide source URLs in your response

## Output Format

When presenting research findings:

```markdown
## [Topic]

### Key Findings
- Finding 1 [Source](url)
- Finding 2 [Source](url)

### Sources
1. [Title](url) - Brief description
2. [Title](url) - Brief description
```
